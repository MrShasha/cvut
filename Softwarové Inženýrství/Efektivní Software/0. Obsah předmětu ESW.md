## Státnicové minimum (ESW)

- **JVM internals:** paměťové rozložení, OOP/compressed OOP, JVM bytecode, JIT (C1/C2, tiered compilation, OSR), safepoints, GC (generational hypothesis, Parallel, G1), profiling (sampling vs. tracing, warm-up).
- **Synchronizace v JVM:** data races, memory barriers, volatile, Java Memory Model, thin/fat/biased locking, CAS operace, atomic field updaters, non-blocking algoritmy (LIFO stack, ConcurrentHashMap).
- **Paměť a datové struktury v JVM:** statická/dynamická analýza, shallow vs. retained size, autoboxing overhead, performance kolekce, open addressing hashing, Bloom filtry, weak/soft/phantom reference.
- **Alokace v JVM:** TLAB (bump-the-pointer), NUMA alignment, escape analysis.
- **Síťová komunikace:** OSI model, C10K problém, blocking vs. non-blocking I/O, select/poll/epoll, Java NIO (ByteBuffer, Channel, Selector), virtual threads.
- **Synchronizace v C/C++:** mutex, semaphore, rw-lock, spinlock — kdy co, jaký overhead. Read-mostly workload, RCU — výhody a nevýhody.
- **Cache-efektivní algoritmy:** SRAM vs. DRAM, typy cache miss, false sharing, MESI protokol, NUMA, cache-oblivious algoritmy, tiled matrix multiplication.
- **Profiling a optimalizace v C/C++:** hardware performance counters (perf), profile-guided optimization (PGO), struktura překladače, LLVM IR, optimalizační průchody.
